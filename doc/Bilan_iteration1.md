# Bilan sur les fonctionnalités :

Nous avons permis au robot d‘utiliser une stratégie aléatoire pour placer une figurine dans un champs
(bois ,pierre ,or et argile)
Nous avons permis au moteur de jeux :
1 - de diminuer le nombre de figurine pour chaque robot (2 robot)
2 - De réduire la capacité d'un champs utilisé par un robot.

# Bilan de l’itération sur les tests :

Nous avons fait des tests pour le moteur de jeux :
1 - Un test pour le choix aléatoire d’un robot.
2 - Un test pour le nombre de figurine de chaque robot avant et après le choix d’un champs.
3 - Un test pour la capacité des champs (bois ,pierre ,or et argile) avant et après le choix d’un robot.
