# Bilan sur les fonctionnalités :

Nous avons permis au robot :
    1- de choisir un nombre aléatoire de figurines à placer.
    2- de lancer de dés.
    3- de récupérer les ressources. (bois ,pierre ,or , argile et nourritures).
Nous avons permis au moteur de jeux :
   1- d'attribuer des ressources au robot.
   2- d'attribuer un score au robot.
   3- de détermine le gagnant en comparant les scores.


# Bilan de l’itération sur les tests :

Nous avons fait des tests pour le Robot :
   1- un test pour le choix aléatoire d'un champs.
   2- un test pour ajouter le champs choisi dans liste de champs (stocker les indice des
champs choisis) .
   3- un test pour mettre à jour le score.
   4- un test pour modifier le nombre de figurines utilisé.

  1- un test pour ajouter un nombre de ressource d'un type choisi .
 Nous avons fait des tests pour le Moteur de jeux :
  2- un test pour la mise à jour de robot.
  3- un test pour la mise à jour de ressources.
  4- un test pour la mise à jour de gagnant.
Nous avons fait des tests pour le dé :
